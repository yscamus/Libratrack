# Libratrack

Libratrack is a web-based reading and resource management system built with Laravel.  
It helps users organize academic and non-academic readings, create personalized lists, and track their reading journey in one place.

## Features
- Manage reading resources
- Create and organize reading lists
- Genre-based browsing
- User dashboard and admin panel

## Built With
- Laravel
- Bootstrap
- MySQL

## Project Status
This project is under active development.
